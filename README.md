## Программы

### Program1 — Базовый синтаксис Си

Демонстрирует базовые конструкции языка в одном файле main.c. Программа просит ввести имя и возраст, определяет категорию по возрасту (if/else if/else), выводит день недели по номеру (switch), читает массив из 5 чисел, считает сумму и максимум (цикл for), делает обратный отсчёт (цикл while), переводит имя в верхний регистр вручную (работа со строками).

### Program2 — Указатели, структуры, файлы

Работа с указателями: изменение значения через указатель, двойной указатель, обмен указателей функцией swap_pointers(int \*\*a, int \*\*b). Структура Student хранит имя, возраст и оценку. Показана цепочка указателей на функции: run_then_show принимает функцию show, которая сама принимает функцию action — три уровня. В конце записывает список студентов в текстовый файл и копирует его в другой файл построчно.

### Program3 — Прикладное программирование

Динамическая память: выделение через malloc, расширение через realloc, освобождение через free. Typedef для указателя на функцию (RecordFunc). Трёхуровневая цепочка: apply_then_show -> run_and_print -> printer. Структура Record пишется в бинарный файл через fwrite (сначала количество записей, потом сами данные блоком), читается обратно через fread в буфер, выделенный через malloc.

### Program4 — Структуры данных и библиотеки

Самый сложный по структуре проект, состоит из нескольких файлов и собирается через Makefile.

**Файлы проекта:**

- `datastruct.h` — заголовочный файл, описывает структуры (Stack, Queue, ListNode) и прототипы функций. Подключается через #include в datastruct.c и main.c. Это и есть интерфейс библиотеки.
- `datastruct.c` — реализация стека (LIFO), очереди (FIFO) и односвязного списка. Компилируется в статическую библиотеку (libdatastruct.a) и в динамическую (libdatastruct.dylib / .so).
- `plugin.c` — отдельный маленький файл с двумя функциями. Компилируется в отдельную .dylib/.so. Нужен только чтобы показать загрузку в рантайме — main.c ничего о нём не знает на этапе компиляции.
- `main.c` — использует структуры данных из библиотеки, и через dlopen/dlsym загружает плагин уже во время работы программы.
- `Makefile` — скрипт сборки. Определяет ОС (Mac или Linux), собирает .o, статическую библиотеку через ar, динамическую через -dynamiclib/-shared, и два итоговых бинаря: program4_static и program4_dynamic.

**Сборка:**
```bash
make clean && make
./program4_static
```

### Program5 — Файлы, каталоги и рекурсия

Работа с файловыми дескрипторами на уровне POSIX (не FILE*, а int fd): open с флагами O_CREAT / O_WRONLY / O_RDONLY, write, read, close. Произвольный доступ к файлу через lseek — перемотка на нужный байт или определение размера файла. Дублирование дескриптора через dup — два fd смотрят на один файл и делят позицию чтения. Рекурсия: факториал, числа Фибоначчи, сумма массива через рекурсию. Работа с каталогами: opendir/readdir/closedir, рекурсивный обход дерева каталогов с отступами по глубине, подсчёт файлов рекурсивно.

### Program6 — Вариадические функции и параметры командной строки

Вариадические функции через stdarg.h: va_list, va_start, va_arg, va_end. Реализованы va_sum (сумма N чисел), va_max (максимум из N чисел), va_print_all (вывод N строк), va_log (логгер с форматом как у printf, внутри использует vprintf для проброса va_list). Параметры командной строки через int main(int argc, char \*argv[]): при запуске программа показывает все аргументы, потом разбирает команду через strcmp и выполняет нужное действие.
